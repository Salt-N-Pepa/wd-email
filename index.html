Utilizando o Editor de Dashboard

O Tutorial a seguir visa demonstrar a funcionalidade de edição de Dashboard que foi implementada no WebDecisor.

Ativando a funcionalidade

Já dentro do WebDecisor, em qualquer uma das visões, você verá no canto superior direito um ícone de lápis (como demonstrado na imagem abaixo). Ao clicar nesse ícone, a funcionalidade será ativada, possibilitando então a edição dos painéis.

Adicionando um painel

Para adicionar um painel que não consta na visão em que você está editando, basta clicar no botão painel que se encontra no canto superior esquerdo. Uma tela com os painéis irá aparecer, basta escolher o painel desejado e clicar em adicionar. Pronto, o painel foi adicionado com sucesso.

Removendo painel

Para remover o painel, basta clicar no botão X no canto superior direito do painel. Uma tela irá aparecer solicitando a confirmação da exclusão. Para excluir basta clicar em Confirmar Exclusão.

Renomeando painéis

Bem como redimensionar, também é possível renomear os painéis, deixando os mesmos de forma mais agradável ao cliente. Com a função de edição ativa, clique no título do painel e uma caixa de edição irá aparecer, possibilitando que o título do painel seja trocado. Para finalizar basta clicar no botão verde para salvar a edição.

Salvando suas edições

Para salvar suas edições, basta clicar no ícone de salvar, no canto superior direito (no mesmo botão que ativa a funcionalidade). Um pop-up irá aparecer confirmando que as alterações foram salvas.
